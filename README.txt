Задание 4 - Визуализация ландшафта

Работу выполнила Глущенко Майя, 322 группа

Запуск: из корневой папки скриптом bash run.sh

Выполнено:
1. Базовая часть:
	визуализация ландшафта с текстурой (по умолчанию и по кнопке 1);
	визуализация нормалей цветом (по кнопке 2);
	визуализация цветом (по кнопке 3);
	освещение.
2. Дополнительная часть:
	– прозрачная вода;
	– имитация волн на воде (реалистичная симуляция волн);
	– туман.

При плохой производительности можно поменять константу drawSeaTime в мэйне для ее увеличения.
